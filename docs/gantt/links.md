Dependency links connect tasks to define their relationships.

## Link Types

- **Finish-to-Start (FS)** — Task B starts after Task A finishes
- **Start-to-Start (SS)** — Both tasks start together
- **Finish-to-Finish (FF)** — Both tasks finish together
- **Start-to-Finish (SF)** — Task A starts before Task B finishes
